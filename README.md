# Valiant Board ✨

Valiant Board est une application moderne et épurée de gestion de tâches (To-Do List) conçue avec un design glassmorphique premium. Elle permet d'organiser vos tâches quotidiennes par catégories, niveaux de priorité et sous-tâches, le tout persistant localement dans votre navigateur.

## 🚀 Fonctionnalités clé

- **Design Premium Glassmorphique** : Interface utilisateur moderne avec des effets de flou d'arrière-plan, des dégradés doux et un mode sombre/clair dynamique.
- **Organisation par Catégories** : Classez vos tâches dans des catégories prédéfinies (Travail, Personnel, Courses, Idées) aux palettes de couleurs soignées.
- **Niveaux de Priorité** : Identifiez rapidement les tâches urgentes grâce aux badges de priorité (Basse, Moyenne, Haute).
- **Sous-tâches interactives** : Divisez chaque tâche en sous-tâches pour suivre précisément votre avancement.
- **Persistance locale** : Vos données sont automatiquement sauvegardées dans le `localStorage` du navigateur.
- **Import/Export JSON** : Sauvegardez et restaurez facilement vos listes de tâches.

## 🛠️ Stack Technique

- **Structure** : HTML5 sémantique
- **Style** : Vanilla CSS 3 (Variables CSS, Flexbox, CSS Grid, Glassmorphism, transitions fluides)
- **Logique** : Vanilla JavaScript (ES Modules, programmation orientée objet avec `TodoService`)
- **Outil de Build** : [Vite](https://vite.dev/)

## ⚙️ Installation et Lancement

1. Installez les dépendances du projet :
   ```bash
   npm install
   ```

2. Lancez le serveur de développement local :
   ```bash
   npm run dev
   ```

3. Ouvrez l'application dans votre navigateur :
   [http://localhost:5173/](http://localhost:5173/)
